# Sprite Factory Pro 1.2.1

Sprite Factory Pro 1.2.1 is a polish and reliability update focused on the redesigned editor shell, Web Sources, workspace handling, batch processing, presets, GIF behavior, and release readiness.

## Highlights

- Refined the main UI geometry so the toolbar, preview studio, workflow dock, export bar, and edit settings panel line up more consistently.
- Reworked Web Sources with saved-page scanning, manual page URL lists, clearer filtering, scan caps, and warnings for very large scan batches.
- Cleaned workspace paging and import presentation so large workspaces remain easier to browse without duplicate import surfaces.
- Improved batch export isolation so copied edits, chosen presets, background overrides, smart preset rules, and file naming behave more predictably.
- Tightened preset handling for mixed sprite/photo/GIF assets and kept preview/export behavior easier to verify before committing changes.
- Restored app icon preference for packaged/runtime launch and refreshed helper guidance to match the current app.

## Notes

- Runtime/build folders remain ignored so the repository stays clean for GitHub.
- Use `docs/RELEASE_CHECKLIST.md` for the practical Windows release pass.
